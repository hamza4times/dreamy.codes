
# Dreamy.codes

Dreamy.codes is a coding platform for people who want to turn ideas into code faster. Dreamy.codes gives you a clean coding environment with open-source free AI-powered assistance right in your browser.

Instead of constantly switching between tutorials, documentation, and chatbots, you can write code and get help in one place.

![Screenshot](assets\screenshot1.png)

### Who it's for

- Students learning to code
- Beginners building their first projects
- Web developers looking for a quick AI coding assistant
- Anyone who wants an easier way to create software

### What you can do

- Write and edit code in your browser
- Chat with AI to generate, explain, and fix code
- Experiment with different AI models (beta)
- Build projects without a complicated setup

Dreamy.codes is built to make coding more accessible, more enjoyable, and a little less intimidating for everyone.
